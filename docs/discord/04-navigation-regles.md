# 📜 Les règles

On fait court, il n'y a pas besoin de plus.

**1️⃣ On reste correct.** Pas d'insultes, pas de harcèlement, pas de discriminations. Zéro tolérance.

**2️⃣ Pas de spam ni de pub.** Ton serveur, ta chaîne, ta boutique, ton bot : demande d'abord à un modo. Partager un outil utile dans une discussion, ça va. Débarquer pour poster un lien, non.

**3️⃣ Pas de contenu illégal.** Vente de proxies, cartes contrefaites, liens douteux, contenu piraté → dehors.

**4️⃣ Le bon salon, le bon sujet.** Ça évite que les vrais retours se perdent dans le flux.

**Modération.** On préfère discuter avant de sanctionner, mais on n'hésite pas à exclure quelqu'un qui pourrit l'ambiance : ce serveur est là pour construire un outil ensemble. Un souci avec quelqu'un ? **Écris à un modo en MP**, c'est traité discrètement.

En restant ici, tu acceptes aussi les [Conditions d'utilisation de Discord](https://discord.com/terms).
