# SEO / Indexation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rendre Wizcard indexable : fondations SEO (robots, sitemap dynamique, manifest, icônes générées, metadataBase + OpenGraph) et crawlabilité des pages dynamiques `decks/[id]` et `users/[userId]` (server shell + generateMetadata + OG dynamiques).

**Architecture:** Fichiers de convention Next.js App Router (`robots.ts`, `sitemap.ts`, `manifest.ts`, `icon.tsx`) à la racine ; root `metadata` enrichi. Les deux pages dynamiques client passent sous une couche server component fine (renommage `page.tsx → *Client.tsx`, nouveau `page.tsx` serveur avec `generateMetadata` + `<h1>` + `opengraph-image.tsx`), avec des fetch **server-safe** dédiés (le fetch deck existant utilise le client navigateur, inutilisable en RSC).

**Tech Stack:** Next.js 16 (App Router, RSC, Metadata API, `next/og` `ImageResponse`), TypeScript, Supabase server client. Pas de nouvelle dépendance.

```ts
import { createClient } from '@/lib/supabase/server';
import type { DeckMeta } from '@/types/decks';
import type { DeckFormat } from '@/types/decks';

/**
 * Server-side deck metadata fetch for generateMetadata / OG / sitemap. Uses the
 * SSR Supabase client (fetchDeckMetaById in deck/db uses the browser client and
 * is unusable in RSC). Public decks are readable by anon via RLS.
 */
export async function fetchDeckMetaServer(deckId: string): Promise<DeckMeta | null> {
	const supabase = await createClient();
	const { data, error } = await supabase
		.from('decks')
		.select('id, owner_id, name, format, description, cover_art_url, created_at, updated_at')
		.eq('id', deckId)
		.maybeSingle();
	if (error || !data) return null;
	return {
		id: data.id as string,
		ownerId: data.owner_id as string,
		name: data.name as string,
		format: (data.format ?? null) as DeckFormat | null,
		description: (data.description ?? null) as string | null,
		folderId: null,
		coverArtUrl: (data.cover_art_url ?? null) as string | null,
		createdAt: data.created_at as string,
		updatedAt: data.updated_at as string,
	};
}
```

```ts
import { createClient } from '@/lib/supabase/server';
import type { Profile } from '@/lib/profile/types';

/**
 * Server-side profile lookup by nickname for generateMetadata / OG / sitemap.
 * The /users/[userId] route param carries the nickname. Profiles are public
 * (RLS SELECT to anon).
 */
export async function fetchProfileByNickname(nickname: string): Promise<Profile | null> {
	const supabase = await createClient();
	const { data, error } = await supabase
		.from('profiles')
		.select('id, nickname, description, avatar_url, created_at, updated_at')
		.eq('nickname', nickname)
		.maybeSingle();
	if (error || !data) return null;
	return {
		id: data.id as string,
		nickname: (data.nickname ?? null) as string | null,
		description: (data.description ?? null) as string | null,
		avatarUrl: (data.avatar_url ?? null) as string | null,
		createdAt: data.created_at as string,
		updatedAt: data.updated_at as string,
	};
}
```

## Task 5: Sitemap dynamique

**Files:**

- Create: `src/app/sitemap.ts`

**Interfaces:**

- Consumes: `SITE_URL` (`@/lib/seo/site`), `@/lib/supabase/server` `createClient`.
- Produces: route `/sitemap.xml`.

**Contexte :** Sitemap dynamique via le client serveur. Statique (landing, search, sets, 3 pages légales) + decks publics + profils. En cas d'erreur DB, retourner au moins les URLs statiques. Nickname nul exclu.

- [ ] **Step 1: sitemap.ts**

Créer `src/app/sitemap.ts` :

```ts
import type { MetadataRoute } from 'next';
import { SITE_URL } from '@/lib/seo/site';
import { createClient } from '@/lib/supabase/server';

export default async function sitemap(): Promise<MetadataRoute.Sitemap> {
	const staticRoutes: MetadataRoute.Sitemap = [
		{ url: `${SITE_URL}/`, changeFrequency: 'weekly', priority: 1 },
		{ url: `${SITE_URL}/search`, changeFrequency: 'weekly', priority: 0.8 },
		{ url: `${SITE_URL}/sets`, changeFrequency: 'monthly', priority: 0.6 },
		{ url: `${SITE_URL}/mentions-legales`, changeFrequency: 'yearly', priority: 0.2 },
		{ url: `${SITE_URL}/confidentialite`, changeFrequency: 'yearly', priority: 0.2 },
		{ url: `${SITE_URL}/cgu`, changeFrequency: 'yearly', priority: 0.2 },
	];

	try {
		const supabase = await createClient();
		const [decks, profiles] = await Promise.all([
			supabase.from('decks').select('id, updated_at'),
			supabase.from('profiles').select('nickname, updated_at'),
		]);

		const deckRoutes: MetadataRoute.Sitemap = (decks.data ?? []).map((d) => ({
			url: `${SITE_URL}/decks/${d.id}`,
			lastModified: d.updated_at ? new Date(d.updated_at as string) : undefined,
			changeFrequency: 'weekly',
			priority: 0.7,
		}));

		const profileRoutes: MetadataRoute.Sitemap = (profiles.data ?? [])
			.filter((p) => p.nickname)
			.map((p) => ({
				url: `${SITE_URL}/users/${encodeURIComponent(p.nickname as string)}`,
				lastModified: p.updated_at ? new Date(p.updated_at as string) : undefined,
				changeFrequency: 'weekly',
				priority: 0.6,
			}));

		return [...staticRoutes, ...deckRoutes, ...profileRoutes];
	} catch {
		return staticRoutes;
	}
}
```
